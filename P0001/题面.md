### 题目名称： 贤者的试炼

### 故事背景：
你是一名魔法学院的学徒，在古老的图书馆中发现了一些破损的魔法卷轴。卷轴上记载着强大的法术，但承载的魔力因年代久远而流失了许多。

卷轴上仅可辨认出两个关键数字：**法则的阶数 $n$** 和 **魔力残存量 $k$**。你的导师告诉你，这个法术的完整形态需要构建一个 $n$ 阶魔法阵。高阶卷轴是 $32$ 阶的，而传奇卷轴则是 $64$ 阶的。

在魔法理论中，$n$ 阶魔法阵可以容纳 $2^n$ 点魔力。为了让卷轴便于使用，导师建议你采用"临界修复法"：不要完全修复卷轴，而是让它保持在**仅差 $1$ 点魔力就能完全恢复**的状态。这样，日后使用时只需注入 $1$ 点魔力就能立即激活法术，非常便捷。

因此，你需要注入的魔力为：$2^n - k - 1$（完整魔力 $2^n$ 减去残存魔力 $k$ 再减去预留的 $1$ 点）。

### 输入格式：
**多组数据**，每行两个正整数 $n$ 和 $k$，其中 $n\in\{32, 64\} , 0 \leq k \lt 2^n$。

### 输出格式：
每行一个整数，表示需要注入的魔力。

### 输入样例：
```
32 2147483647
```
### 输出样例：

```
2147483648
```

### 样例解释：

对于 $32$ 阶魔法阵，完整魔力为 $2^{32} = 4294967296$ 点。当前残存 $2147483647$ 点魔力，采用临界修复法需要注入：$2^{32} - 2147483647 - 1 = 2147483648$ 点魔力。修复后卷轴将有 $2147483647 + 2147483648 = 4294967295$ 点魔力，仅差 $1$ 点即可达到完整状态。
